C++使用语言

一、已掌握的核心知识 截至 L2-3

变量与类型 掌握

会用 int（整数）、string（字符串）装数据。

int a, b; // 整数 string key; // 字符串
输入与输出 掌握

cout<< 输出、cin>> 输入、endl 换行；懂多变量输入顺序（先写的先拿到）。

cin >> a; cin >> b; // 输18再输5→a=18,b=5
if-else 语句 掌握·重点

按条件成不成立执行不同代码，只走一边。会用 == > >= <。

if(a>10){ cout<<"s"; } // 成立 else { cout<<"n"; } // 否则
进阶 / 易错点 考点
  • 两个并列 if 可能都执行；if-else 只执行一个，只有条件"正好相反"才能互换。
  • 三个坑：if 条件后多分号、cout 漏分号、else 后加条件。
舵机开门 rotateTo() 掌握

括号填角度 0~180°，舵机带门转。

if(isPressed("A")){ rotateTo(180); }
红外遥控 getIRKey() 掌握

读"按了哪个键"存进变量。

key = getIRKey(); if(key=="左"){ rotateTo(180); }
记录密码 拼接+isdigit() 掌握

+ 把数字一个个接起来（"12"+"3"="123"）；isdigit() 判断是不是数字。

if(isdigit(key)) result = result + key;
验证密码 if-else 实战 掌握

== 比对，对了开门等5秒自动关，错了提示。

if(result==password){ rotateTo(180);sleep(5);rotateTo(90); }else{ printPos("密码错误"); }

🔐 能独立做出的项目：密码门 / 密码锁

遥控输入密码 → 记录拼接 → if-else 验证 → 正确开门、5 秒自动关，错误提示。综合用到字符串、输入检测、条件判断、硬件控制，是完整的"输入→处理→输出"小系统，已能动手完成并邀请家人猜密码。

三、正在学习 L2-4 · 已预习，课堂进行中

赋值语句 学习中

a=5 直接放；a=a+2 分两步：先算右边、再放进左边。

a = 5; // 直接放 a = a + 2; // 先算a+2再赋值
变量控制循环 学习中

用变量代替数字当循环次数，次数随变量变。

for(int i=1;i<=a;i++){ cout<<"f"; } // 循环a次
变量命名规则 学习中
  • 区分大小写；只用字母/下划线/数字、不能数字开头；不能用关键字（if、int）。
