二、已掌握的核心知识 截至 L2-3

① 编程基础

变量与数据类型 已掌握

会用 int（整数）和 string（字符串）两种变量来"装"数据。

int a, b; // 整数变量 string key; // 字符串变量
输入与输出 已掌握

用 cout << 输出、cin >> 输入，endl 换行。理解多变量输入的顺序：先写的变量先拿到数字。

cin >> a; cin >> b; // 先输 18 再输 5 → a=18, b=5 cout << a << endl;

② 条件判断（本阶段重点）

if-else 语句 已掌握·重点

根据条件成不成立执行不同代码，只走一边，绝不两边都执行。会用比较运算符 ==、>、>=、<。

if(a > 10) { cout<<"s"; } // 成立打印 s else { cout<<"n"; } // 否则打印 n
进阶与易错点 考点
  • 两个并列 if vs if-else：两个 if 可能都执行；if-else 只执行一个。只有条件"正好相反"时才能互换。
  • 三个常见错误：① if 条件后多写分号 if(a<3;)　② cout 句漏分号　③ else 后多加条件 else(a>=3)

③ 智能硬件 ·「密码门」项目（L2-3）

用智慧小屋 + 舵机 + 红外遥控器，把编程和真实硬件结合，做出一个会开关的密码锁。

控制舵机开门 — rotateTo() 已掌握

括号里填角度（范围 0~180 度），舵机带着门转。

if(isPressed("A")) { rotateTo(180); } // 按A键，转到180°开门
读红外遥控 — getIRKey() 已掌握

读出"刚按了遥控的哪个键"，存进字符串变量。

string key; key = getIRKey(); // 检测按键 if(key == "左") { rotateTo(180); }
记录密码 — 字符串拼接 + isdigit() 已掌握

用 + 把按下的数字一个个接起来（"12"+"3"="123"）；用 isdigit() 判断按的是不是数字。

if(isdigit(key)) // 按的是数字吗 { result = result + key; } // 拼接到密码后面
验证密码 — if-else 实战 已掌握

用 == 比较输入与正确密码，对了开门、等 5 秒自动关；错了屏幕提示。把学过的 if-else 用在真实项目里。

if(result == password) { rotateTo(180); sleep(5); rotateTo(90); // 开门→等待→关门 } else { printPos("密码错误", "中间"); }

四、正在学习 L2-4 · 已预习，课堂进行中

模块 15「赋值语句初识」+ 模块 16「简单变量运算」。核心是让一个变量（盒子）里的数值变来变去。

赋值语句 学习中

a = 5 直接赋值；a = a + 2 要分两步：先算右边算式，再把结果放进左边变量。

a = 5; // 直接把 5 放进 a a = a + 2; // 先算 a+2，再赋值给 a
变量控制循环次数 学习中

for 小括号中间的数字是循环次数；用变量代替数字，次数就随变量变化。

for(int i=1; i<=a; i++) { cout<<"f"; } // 循环 a 次
变量命名规则 学习中
  • 区分大小写（a 和 A 不同）；只能用字母、下划线、数字，不能数字开头；不能用 C++ 关键字（如 if、int）。
