# 学习记忆 · 成长记录

> 这是「学习记忆层」的人类可读门面。机器读的结构化数据在 `data/learning/`，
> 这里是给爸爸随时翻看的成长档案。**每次出卷/批改后，AI 会在此追加一段。**

## 这套机制是怎么运转的

AI（Claude Code）本身没有跨会话记忆，每次会话都是把仓库重新克隆一份。
所以「长期记忆」靠的是**仓库里的文件**——它们就是 AI 的大脑。运转闭环：

```
出新内容前  →  读 data/learning/（学情画像 + 卷子台账）+ 本文件
              ↓  基于历史决定考什么、难度多少
生成卷子/软件  →  放入 papers/ 、apps/
出完之后    →  更新 papers_log.json（记一条）、profile_*.json（调掌握度）、本文件（追加一段）
              ↓  commit 进仓库 = 记忆固化，下次会话能读到
```

只要这条闭环每次都走，内容就会「根据以前的内容迭代进化」。
