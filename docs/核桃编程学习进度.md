## 三、已掌握知识点（按课次）

### L1-2：if 语句 / 变量 / 关系运算符

- **if 语句**：条件控制语句，用于判断
  - 条件成立 → 执行大括号 `{}` 内代码
  - 条件不成立 → 跳过
  - 结构：`if (条件) { 代码 }`
- **单 if 语句**：只有一个判断
- **并列 if 语句**：多个 if 一个接一个，从上到下依次判断，互相之间无数理关系
- **变量**：带名字的容器（盒子），值可变，新值覆盖旧值
  - 使用三步：`int a;`（定义）→ `cin >> a;`（输入）→ `cout << a;`（输出）
- **关系运算符**（6 个）：`>` 大于、`<` 小于、`==` 等于、`>=` 大于等于、`<=` 小于等于、`!=` 不等于
  - 注意：判断"等于"用 `==`，不是 `=`
- **关系表达式**：写在 if 括号里的判断条件，如 `b>5`、`a<7`、`b==7`

### L1-3：for 语句（核心）

- **for 语句作用**：重复执行某段代码（循环结构）
- **语法格式**：`for (变量初始化; 循环条件; 步长) { 重复执行的代码 }`
- **for 三要素**（重点）：
  - `int i=1`：循环初始化，开始循环前新建变量 i 并赋初始值
  - `i<=7`：循环条件（重点中的重点），每次重复前判断 i 是否符合条件，符合就执行
  - `i++`：步长，每次重复后 i 增加 1（i 每次变动的值）
- **执行流程**：初始化 → 判断条件 → 成立则执行代码 → 步长 → 回到判断条件 → 不成立则结束
- **顺序执行中的 for**（重难点）：
  - 从第一行代码开始，依次向下读，直到 `return 0` 结束
  - 遇到 if：看条件，成立才执行大括号内代码，然后继续向下
  - 遇到 for：先看大括号要重复什么，再看小括号定的重复次数
  - 示例代码：
    ```cpp
    int a;
    cin >> a;
    if(a<5) { cout << "g" << endl; }
    cout << "tr" << endl;
    for(int i=1; i<=3; i++) {
        cout << "f" << endl;
        cout << "d" << endl;
    }
    ```

### L1-4：智能门铃 & 语音留言（指令类）

- **检测按钮**：
  - `isPressed("A")`：判断按钮是否被按下，括号内填按钮名称，名称要加双引号
  - `playSound("ring")`：播放声音，括号内填声音名称，要加双引号
  - 示例：`if(isPressed("A")) { playSound("ring"); }`
- **打印文字**：
  - `printPos("文字")`：在智慧核心左上角打印文字
  - `printPos("内容", "位置")`：在指定位置打印（前=内容，后=位置，如"中间"）
- **检测触摸传感器**：
  - `isTouched()`：有人按下→条件成立，反之不成立
  - 示例：`if(isTouched()) { printPos("按下"); }`
- **访客留言**：
  - `record()`：录制声音
  - `playSound()`：播放声音

### L2-1：逻辑运算符 && / ||（核心 + 赛考重点）

- **逻辑与 `&&`（与）**：
  - 规则：左右两边条件**都成立**，整个条件才成立
  - 口诀：与要"都"
  - 示例 `if(a>10 && b>10)`：
    - a=12,b=20 → 成立
    - a=3,b=20 → 不成立
    - a=9,b=9 → 不成立
    - a=11,b=9 → 不成立
- **逻辑或 `||`（或）**：
  - 规则：左右两边只要**有一个成立**（一个或两个），整个条件就成立
  - 口诀：或要"有"
  - 注意：①其中一个条件成立则整体成立 ②两个都不成立则整体不成立
  - 示例 `if(a>10 || b>10)`：
    - a=17,b=18 → 成立
    - a=7,b=15 → 成立
    - a=7,b=8 → 不成立
    - a=11,b=9 → 成立
- **数学连接**（哥哥重点）：
  - 编程 `&&` ↔ 数学"且"（∧）
  - 编程 `||` ↔ 数学"或"（∨）
  - 应用：不等式组（如 x>3 ∧ x<5）、几何证明条件组合（如"平行四边形 且 有一个直角"是矩形）

## 四、L2-2 预习要点（即将学，尚未掌握）

- **课程主题**：if-else 语句初识 & 进阶
- **配套项目**：用编程做月相观测仪（输入农历日期，舞台显示对应月相+科普）
- **学习目标**：
  1. 复习多个 if 顺序执行的执行逻辑（重点）
  2. 认识 if-else 语句，掌握运行逻辑及语法结构（重点）
  3. 辨析两个 if 组合 vs if-else，知道何时可改写（难点）
  4. 连续输入多个数据时，按 cin 顺序确定变量顺序
- **if-else 核心**：如果条件成立走 if 分支，否则走 else 分支，只走一边（非此即彼）
  ```cpp
  if(score >= 60) {
      cout << "及格" << endl;
  } else {
      cout << "不及格" << endl;
  }
  ```
- **两个 if vs if-else**：当两个 if 条件正好相反时（如 >=60 和 <60），可改写为 if-else
- **cin 输入顺序**：`cin >> a >> b;` 先写的变量先拿到数字
- **赛考重点**：选择结构（if / 并列 if / if-else）、变量使用、单个/多个变量输入输出
