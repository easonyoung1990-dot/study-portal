# 外教课（Preply）学情分析与短期规划

> 用途：跟踪外教课的**教学逻辑、孩子进步、效果与效率**，并据此定短期目标、指导每期复习软件出题。
> 数据源：`docs/PREPLY_REVIEW_APP_SPEC.md` 第六节（5 节课内容明细）、`data/preply.json`、`data/learning/profile_*.json`、`data/learning/english_eddey.json`。
> 维护：每收到一节新课报告，更新「老师评价趋势」与「老大难清单」两张表。

## ⚠️ 重要前提（结论别高估）
1. 目前只有归档的 **5 节课**数据；孩子若学得更久，更早的数据不在仓库里。
2. Preply 课堂报告是「**课堂见解 · AI 测试版**」自动生成的（报告自带免责声明"AI 生成的见解可能存在错误"）。**发音准确率 %、纠错均为 AI 估算，看趋势可以，别当精确分数。**
3. 这 5 节报告**未按 Kenton / Eddey 分开**，是合并画像。按校内英语档案曾推测 Kenton 偏强；但 **2026-06-23 老师 Denys 已正式回信：两个孩子水平其实差不多（见下「八」）**——以老师口径为准，别再高估两人差距。

## 一、老师的教学逻辑（5 节课走向）

| 顺序 | 主题 | 核心语法 | 拔高词(B1/B2) |
|---|---|---|---|
| 1 | 情绪 Feelings | I feel… when… / He is… because… | — |
| 2 | 学校科目+日常 | 现在进行/习惯 we have / are having | — |
| 3 | 描述房间房子 | **There is / There are** + 复数 | mirror, verify |
| 4 | 房间+好玩的活动 | There is/are 再练 + 楼层 + garage/car park | — |
| 5 | 旅行交通+建议 | **should/shouldn't** + ⏰**一般过去时**(首次扩展时态) | transport, supercar |

**逻辑要点：**
1. **主题从"身边具体"往外扩**：情绪 → 学校 → 家 → 房间里玩 → 出门交通旅行。都能调动生活经验、能说出口，不抠语法书。
2. **螺旋式重教**：房间/There is-are 在第 3、4 课连上两次——老师咬住一个点反复练。
3. **语法阶梯清晰**：描述句 → 进行/习惯 → 存在句+复数 → 情态建议(should) → 过去时。到第 5 课才动时态，稳扎稳打。
4. **每课 1–2 个超纲词**当"甜点"，扩上限不作主线。
5. **沟通式 + 即时纠错(recast)**：报告全是"你说错的→应该说的"成对出现。

## 三、有没有效果？——有，但有"老大难"

**✅ 有效的证据**
- "表现良好"从 0 → 5 个；
- 早期教的点开始内化：冠词 `a` 在第 5 课从"错点"变"夸点"（a bus driver 对、a busman 还错）= 正在巩固、尚不稳定；
- 发音可被刷出来：Eddey 档案里 `swimming` 练 **82 次** → 课堂 69%、整体正确率 95%。**针对性重复管用。**

**⚠️ 老大难清单（反复错 = 没真正消化，每期软件固定复练）**

| 老大难 | 出现的课 | 病根 | 目标 |
|---|---|---|---|
| 名词复数漏 s | 3 + 4 | bedroom→bedroom**s** 连错两课 | 数字/复数主语后名词必加 s，稳定 90%+ |
| 冠词 a/an/the | 3 + 5 | a holiday / a busman / the bus | 职业/单数名词前 a；特指 the |
| 不地道（中式直译） | 4 + 5 | 用 use/have 套"用/有"→ 应 take | take the bus / take classes |
| 介词 by/in/on | 1 + 5 | in English class / by car | 交通 by；场所 in |
| 长词发音(多音节) | 2–5 | 单音节熟、长词乱 | <50% 的词刷到正确重音 |

## 四、学习效率

- **课堂效率：高**——沟通式、即时纠错、量化反馈，孩子敢说，优质外教。
- **瓶颈在课与课之间，不在课上**：一周一节≈低频曝光，"老大难"跨月回潮 = 遗忘曲线没被打断、留存不足。
- **撬动杠杆已验证**：swimming 82 次刷到掌握，证明"把老师纠的错 → 转成课前 5 分钟反复练"有效。现有 Preply 复习软件（错题关+发音关+课前冲刺）正对症。**坚持课前刷 = 把"周频"补成"日频"。**

---

## 五、短期目标（暑期 6–8 周，频率提高后）

> 暑假上课频率增加，是把"周频补成日频"、专攻老大难的窗口。

- **G1 复数 -s 自动化**：数字/many/复数主语后名词必加 s（rooms/bedrooms/passengers）→ 稳定 90%+。
- **G2 冠词 a/an/the**：职业/单数名词前 a（a bus driver）、特指 the（the bus）→ 从"对错参半"到稳定。
- **G3 地道动词搭配**：take the bus / take classes / go by… 取代 use/have 直译。
- **G4 介词**：交通 by、场所 in（by car / in English class）。
- **G5 一般过去时**：高频不规则 went/ate/took/made + 规则 -ed，能完整讲"昨天做了什么"。
- **G6 长词发音**：把上面 <50% 的多音节词刷到正确重音。
- **能力目标**：敢用完整句开口，减少中式直译；每次能主动说出 2–3 句"超纲"长句（如现在完成时）。

## 六、我们怎么配合外教
1. **课前必刷 5 分钟冲刺**（效率关键动作）：用按本课报告定制的复习软件，专练老师纠的错 + 发音弱词。
2. **请老师持续"回收"老大难**：即便主题变了，也顺手再纠复数 s / 冠词 / take 类动词——帮孩子练成肌肉记忆。
3. **请老师把"纠错句"和"发音弱词"写具体**：报告越具体，我们在家练得越准。
4. **每期新课**：按 `PREPLY_REVIEW_APP_SPEC.md` 工作流出软件，并回填本文件两张趋势表。

## 八、老师分孩子评价 + 教学理念（Denys 回信，2026-06-23）

> 这是 Denys 老师对第七节「反馈信」里问题（两个孩子分别怎么样 / 水平 / 该练什么）的正式回复。
> **比之前"合并 AI 报告 + 校内档案推测"更权威，今后按这个来。** 出英语软件/内容前先读这一节。

### 1. 对"错误"的看法（重要 · 直接决定我们怎么练）
- 两个孩子犯的错都是**这个阶段非常正常**的：有的是某个结构还没学到，有的是**为了流利、想快点把意思说出来**而顺带出的错。
- 老师理念：这个阶段 **流利度(fluency) 通常比准确度(accuracy) 长得快**；只要有足够输入和练习，**准确度会慢慢追上来**。
- 学校往往要求"每句都对"；但真实交流里，**敢说、能把想法组织出来、能把对话接下去**往往更重要。
- → **我们的对策（写进英语软件设计）**：
  - 英语软件**不要做成"纠错罚站 / 抠每句对错"**；纠错要**轻、要鼓励**、即时正面反馈。
  - 多给"**敢开口、能表达**"的正向激励；错了把"为什么"讲清楚就好，**别让孩子怕错**。
  - 口语/对话类任务重点放在"**把话接下去、把意思讲完整**"，不是逼到零语法错误。

### 2. 两个孩子的水平 & 课堂表现
- **水平：两人差不多（about a similar level）。** → **修正**之前"Kenton 明显更强"的推测。内容/难度可两人通用，仅按各自校内年级（Kenton 四下 / Eddey 三下）微调。
- **课堂表现：都很好带、都很棒。**
  - **Kenton 🦁**：全程更**专注**、稳。
  - **Eddey 🐯**：偶尔会**走神一小会儿**，但不严重，**很快能拉回来**、继续好好参与。
- → 给 Eddey 的软件多用**短回合 + 强反馈 + 会动的交互**（本就符合他"视觉化/短闯关更有效"的画像），帮他保持注意力。

### 3. 老师的教学方法：可理解性输入（i+1）
- 老师会**用略高于他们当前水平**的语言跟他们说话、提问（comprehensible input / i+1）。
- 即使他们**暂时说不出**这些新词 / 新句型，**反复听、反复处理**会建立熟悉感，**慢慢就自然用出来**——让语法变得**自动**，而不是每次都要刻意想。
- → **我们的对策**：复习软件里多放**略超纲的自然例句 / 地道搭配**让他们"磨耳朵"（真人朗读 + 长按看中文），**不要求立刻产出**；这正是"会动的图 / 分步动画 / 朗读"承担的"输入"角色。

### 4. 暑期重点（老师确认 · 两娃通用）
老师确认：继续打磨我们已识别的反复点，**跨不同主题反复回收**，越练越自然——
1. **冠词 a / an / the**
2. **名词复数 -s**
3. **自然表达 & 固定搭配**（natural expressions / collocations，如 take the bus、go on vacation）
4. **介词 prepositions**（by car / in class / on holiday）
5. **动词时态形式 verb tense forms**

> 与第五节「短期目标 G1–G6」完全一致，继续沿用；新增的总基调是 **"流利优先、轻纠错、多输入(i+1)"**。
