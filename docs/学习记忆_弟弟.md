# 三、三科共同根因链

弟弟最近三科的问题可以统一归纳为一条链：

```text
读题不精准 → 关键词没圈出 → 调错规则/漏规则 → 中间量或单位丢失 → 答案表达不完整 → 做完不检查
```

这条链在三科里的表现不同：

| 学科 | 外在表现 | 底层原因 | 补救方向 |
| --- | --- | --- | --- |
| 语文 | 字词不精确、课文只记大概、阅读题找不到证据、主观题答不满。 | 没有形成“回文定位+分点作答+原句积累”的考试习惯。 | 字词默写、课文细节、阅读三步法、转述和作文骨架。 |
| 数学 | 计算错多、余数单位错、应用题不会建模、面积组合题乱。 | 计算规则、单位量感、图形路径、题意建模没有自动化。 | 竖式/除法/单位/面积/应用题母题，每次带一题一类题。 |
| 英语 | is/are、单复数、this/these、do/does/can 后动词、完整回答反复错。 | 主语数量和句型规则没先判断，看到词就套。 | 先判断“一个/很多、谁做主语、有没有 can/do/does、问什么类别”。 |

# 四、语文档案

## 4.1 语文总体薄弱点

- 字词：容易“会认但写不准”，错在字形细节、同音近形、词语搭配。
- 课文：能说大意，但考试常考“原句、细节、顺序、原因”，弟弟掌握不够精确。
- 阅读：不会稳定回到原文找证据，容易凭印象答。
- 主观题：不会按“两点/三点”分点回答，句式不够完整。
- 作文：结构、细节、结尾升华不足，语言不够考试化。
- 专项“转述”：人称、时间词、指示词、标点转换不稳。

## 4.3 转述专项核心模板

例题：爸爸说：“我明天要去昆明，你自己做饭吃吧。”

正确思路：

- 第一步：去掉冒号和引号。
- 第二步：“我”要看是谁说的，这里是爸爸，所以改成“他”。
- 第三步：“你”要看爸爸对谁说，这里是对“我”说，所以改成“我”。
- 第四步：“明天”如果题目没有要求改时间，可以保留；若转述发生在第二天，则要改成“第二天”。小学常规题先按原句逻辑保留。

标准：爸爸说明天他要去昆明，让我自己做饭吃。

弟弟要记住的口诀：先找说话人，再换我你他；冒号引号去掉，语气词改成“让/叫/告诉”。

## 4.4 语文后续复习优先级

| 优先级 | 内容 | 训练方式 |

| A | 期末复习索引里的字词、课文、古诗。 | 每日 8-10 个词 + 1 个课文/古诗填空 + 次日回炉。 |

# 五、数学档案

## 5.1 数学总体判断

数学核心结论：弟弟不是某一道题不会，而是底层规则没有自动化。简单口算、直接套公式的题能做；一旦题目多一步、要换单位、要找中间量、要画图，就容易断。

# 六、英语档案

## 6.1 英语总体薄弱点

- 单复数链条：this/that/these/those、it/they、is/are 联动不稳。
- 主语判断：he/she/it/人名作主语时，动词三单容易漏；does 后动词又容易忘记还原。
- can/do/does 后动词原形：看到动词就乱加 s 或 ing。
- 特殊疑问句混淆：What colour / What shape / What season / How many / How much 容易看错问法。
- 完整回答弱：阅读题和问答题容易只写半句或漏主语、漏 be 动词。
- 不乱加 the：名词前是否要 the 需要按句意判断。

## 6.2 已归档的英语错题/问题来源

| 时间/来源 | 错题/问题 | 根因 | 后续训练 |
| --- | --- | --- | --- |
| 2026-04-22 语法考前弱项卡 | is/are；this/that/these/those；What is this?/What are these?；完整回答；单复数改句；不乱加 the；小阅读找信息。 | 基础句型链条没有自动化。 | 先判断“一个还是很多”，再选 this/these 和 is/are。 |
| 2026-05-29 英语薄弱报告 | 单复数 / this-these / it-they / is-are；do/can 后动词原形；特殊疑问句分类。 | 句子开头没判断主语和问法。 | 家长带练口令：一个还是很多？题问什么类别？有没有 can/do/does？ |
| 2026-06-08 课外英语复习 Module 2 | Animals / Toys / Clothes；Do you like elephants? Does he like robots? She likes tigers. What do you like? What are these? They’re gloves. What’s this/that? It’s a scarf. | 一般现在时、三单、单复数、完整回答都需要巩固。 | 造句：用三单造句，并改成否定句、一般疑问句、特殊疑问句。 |
| 2026-06-09 老师反馈 M1复习 | do/does 疑问句；三单感官动词 How does it feel?；How many/How much；作文重写。 | does 后动词还原、how 类问句分类不稳。 | 错题整理除听力外全部回炉；作文按模板重写。 |
| 长期阅读问题 | 阅读回答写不完整。 | 没有形成“找关键词→抄准信息→完整句回答”的步骤。 | 每次阅读只练2题，但必须完整句。 |

## 6.3 英语核心口令

| 场景 | 弟弟先问自己 | 正确规则 |
| --- | --- | --- |
| 单复数 | 是一个，还是很多？ | 一个：this/that/it/is；很多：these/those/they/are。 |
| 一般疑问句 | 有没有 do/does/can？ | do/does/can 后面动词用原形。 |
| 三单 | 主语是不是 he/she/it/人名？ | 普通陈述句动词加 s/es；但 does 后不加。 |
| 特殊疑问句 | 题目问颜色、形状、季节、数量、价钱，还是东西？ | What colour / What shape / What season / How many / How much / What。 |
| 完整回答 | 我有没有写主语和动词？ | 不能只写一个词，要写完整句。 |

# 八、Claude 复习软件总规格

## 8.1 软件定位

软件不是普通刷题器，而是“错因诊断 + 短时复习 + 隔天回炉”的儿童互动系统。每次使用控制在 15 分钟内。

## 8.2 必须有的模块

| 模块 | 功能 |
| --- | --- |
| 首页 | 显示今天三科任务：语文15分钟、数学15分钟、英语15分钟。 |
| 错因标签库 | 每题错后自动显示：读题错/规则错/单位错/计算错/表达不完整/没检查。 |
| 语文闯关 | 字词默写、课文填空、古诗默写、转述、阅读回文定位。 |
| 数学闯关 | 计算、余数单位、单位换算、面积、应用题、分数、统计图。 |
| 英语闯关 | 单复数、is/are、do/does/can、三单、特殊疑问句、完整回答。 |
| 动画讲解 | 数学重点题必须有图形/线段图/实物动画，尤其余数还原、面积拆补、鸡兔同笼/假设调整。 |
| 同类变式 | 每道错题后必须给1道变式，确认不是背答案。 |
| 隔天回炉 | 错题自动进入明天“5分钟回炉”。 |
| 家长看板 | 显示三科弱点、连续错因、建议下一步。 |

## 8.3 每题交互模板

```text
题目展示 → 第一步提示：圈关键词 → 第二步提示：选规则 → 学生作答 → 即时判断 → 错因标签 → 小学生能懂讲解 → 同类变式1题 → 加入/移出错题本
```

## 8.4 数学动画重点

- 余数还原：800÷90 → 80÷9，说明 80 是“80个十元”，余8就是8个十元=80元。
- 组合图形面积：拖动辅助线，把大图拆成两个长方形，或补成大长方形再减掉小块。
- 线段图：用颜色显示“少的一份”和“多出来的量”。
- 速度时间路程：三角关系图：路程=速度×时间，速度=路程÷时间，时间=路程÷速度。
- 统计图：把睡觉时间转成分钟或时间轴，生成条形图，并写“最多/最少/建议”。
