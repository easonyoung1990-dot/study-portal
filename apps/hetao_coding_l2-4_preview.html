🌳 知识点 1 · if-else 分岔
方法点拨：if-else 像一道分岔路口。先看条件成不成立：成立走 if 这边，不成立走 else 这边，只能走一边，绝不会两边都走。
if(a > 6)
{ cout<<"g"; } // 条件成立时
else
{ cout<<"k"; } // 不成立时
a = 5，条件 a > 6 不成立
拖动滑块改变 a，再点「让小球走一遍」，看它走哪条路。
🔁 知识点 2 · 变量控制循环次数
方法点拨：for 小括号中间那个数字，就是循环次数。如果次数不确定，可以用变量代替数字——次数就会跟着变量一起变。
for(int i=1; i<=3; i++)
{ cout<<"f"; } // 重复打印 f
循环次数 a = 3 次
点上面按钮，看打印几个 f
把次数从写死的 3 换成变量 a，就能让同一段代码跑不同的遍数——这就是"变量控制循环"。
📦 知识点 3 · 赋值语句魔法盒
方法点拨：赋值语句就是改变左边盒子里的数。等号右边是算式时，先算出结果，再放进左边的盒子。所以要分两步看：①算右边 → ②放进左边。

✏️ 知识点 4 · 变量命名规则
三条规则：
① 区分大小写：a 和 A 是两个不同的变量。
② 只能用字母、下划线 _、数字 0–9，且不能数字开头。
③ 不能是 C++ 关键字（如 if、int 这些有特殊含义的词）。
