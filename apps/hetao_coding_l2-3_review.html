🔐 智能密码门，我们做了什么？
上节课我们用「智慧核心」做了一个真能用的密码门。这个实验室把每一步都做成会动的，点一点、拖一拖，把"为什么"看清楚——不是再做一遍题。
🧩 四个核心零件：① 舵机 rotateTo 开关门 · ② 红外遥控 getIRKey 读按键 · ③ 字符串拼接 记密码 · ④ if 判断 验证密码。下面一个个玩。

🔧 舵机实验室 · rotateTo()
舵机就是门上的"小马达"。rotateTo(角度) 里填一个数字，代表转到第几度。范围 0°～180°。拖一拖，看门怎么转。
90°
rotateTo(90); // 关门
💡 记住：括号里只能填 0 到 180 的数字。项目里我们让门：开 rotateTo(180)，关 rotateTo(90)。
📡 红外遥控 · getIRKey()
getIRKey() 的作用：读出你刚按了遥控的哪个键，把这个键的名字交给程序。按一按下面的遥控，看屏幕。
📡
string key;
key = getIRKey();
智慧核心读到的 key： （按一下遥控）
💡 key = getIRKey(); 就是把"刚按的键名"装进字符串变量 key 里，后面就能判断它是不是密码了。
🔑 记录密码 · 字符串拼接 + isdigit
密码是好几个数字。我们要把按下的数字一个个接起来变成一串。靠 + 拼接，靠 isdigit() 检查"按的是不是数字"。
isdigit("12") → 成立 ✅（全是数字）　isdigit("ab") → 不成立 ❌（不是数字）
if( isdigit(key) ) // 按的是数字吗？
    result = result + key; // 接到密码后面
result（已记下的密码）：
（空）
按数字会被接进 result；按字母 A，isdigit 不成立，不会被记进去。
🔐 验证密码 · if(result == password)
用 == 判断"输入的密码"和"正确密码"是否一样。一样就开门，不一样就报错。先设个密码，再用遥控输入试试。
string password = "123"; // 正确密码
if(result == password)
{
  rotateTo(180); // 开门
  sleep(5);  // 等 5 秒
  rotateTo(90);  // 关门
}
else
{
  printPos("密码错误");
}
正确密码（点数字设置）：123
我输入：（空）
设好密码，输入后点「验证开门」。输入和正确密码完全一样才会开门哦。
